🚀 Excited to share my latest Data Science & Predictive Analytics Project!

I recently built an end-to-end E-Commerce Customer Purchase Analysis System from scratch. The goal was to transform raw transactional data into actionable business intelligence and deploy predictive models to forecast customer behavior.

Here is a breakdown of the comprehensive pipeline I engineered using Python, Pandas, NumPy, Scikit-Learn, Seaborn, and Matplotlib:

🔹 Data Preprocessing & Cleaning: Managed inconsistent column names, implemented robust imputation strategies (using medians for skewed numerical columns and modes for categorical distributions), and handled time-series alignment via forward/backward fills.

🔹 Feature Engineering & Sentiment-Driven Simulation: Modeled core purchasing metrics like Total Revenue. To satisfy advanced project constraints where explicit customer feedback was absent, I implemented a randomized simulation to dynamically generate Rating matrices aligned with transaction distributions.

🔹 Exploratory Data Analysis (EDA): Aggregated metrics at the product level (Product_SKU) to extract cumulative revenues, average pricing structures, and ceiling prices. I identified top-selling items and optimized inventory insights by isolating geographic purchase concentrations.

🔹 Data Visualization: Developed high-impact visual components including:

A localized product revenue analysis bar chart.

A clutter-free Product Category distribution chart utilizing an externalized percentage legend to completely prevent text overlapping.

A demographic/geographic breakdown of customers by city mapped against gender.

🔹 Machine Learning (Predictive Modeling): Built and trained a Multiple Linear Regression framework to predict customer total spending. Selected key predictive variables including product pricing architectures, purchase quantities, and user ratings. The model was trained using an 80/20 train-test split, evaluating performance using Mean Absolute Error (MAE) and $R^2$ Score metrics to validate its accuracy before deployment.

💡 Key Business Insights Extracted:

High-Value Driving Categories: Identified the precise product category driving maximum revenue to help focus marketing budgets.

Geographic Sweet Spots: Isolated the exact cities holding the largest consumer footprint, allowing for highly targeted regional campaigns.

The Revenue Drivers: Proven mathematically how incremental shifts in price-point choices and quantity demands dictate total customer spending.

This project was a fantastic exercise in balancing rigorous data hygiene with statistical modeling to answer real-world e-commerce problems.
